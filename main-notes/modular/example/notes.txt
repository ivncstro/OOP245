Preprocessor: Interprets all directives creating a single translation unit for the compiler 
- (inserts the contents of all #include header files), (substitutes all #definemacros)

Compiler: Compiles each translation unit separately and creates a corresponding binary version

Linker: Assembles the various binary units along with the system binaries to create one complete 
executable binary
